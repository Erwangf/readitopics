Readitopics configuration manager

On this app, you can manage dataset and configurations.

Dataset

A dataset is composed as one or more text files, used by Readitopics as text corpus. It can be :

  • one simple file (ex : myFile.txt)
  • many files (ex: book1.txt, book2.txt, book3.txt)
  • multiple folders of files

To use a dataset, you need a configuration. Many configuration cas be used with one dataset.

Configuration

Chaque configuration contient plusieurs paramètres nécessaires aux opérations de Readitopics, tels que le nombre d'itération pour la LDA, le nombre de threads à utiliser, le prétraitement, ainsi que le dataset associé à la configuration.
